Manual para la ejecucción del Proyecto final de Python Intermedio AM 

Pasos:
1-Abrir el IDE: 
Abre el IDE( por ejemplo Visual Studio Code) en tu computadora.

2-Abrir el archivo del código: 
Descarga el archivo zip, guardalo en el directorio de su preferencia y ahí mismo
descromprima ese archivo.

Si el archivo del código Python ya ha sido creado, ábrelo haciendo clic en "Archivo" 
en la barra de menú, luego selecciona "Abrir archivo" y busca el archivo que contiene 
el código Python (por ejemplo, "script_crud.py"). Haz doble clic para abrirlo en el 
editor.

3-Localizar la terminal o consola: 
En Visual Studio Code, asegúrate de que la terminal integrada esté visible en la parte 
inferior del IDE. Si no ves la terminal, ve al menú "Ver" y selecciona "Terminal 
Integrada" para mostrarla.

4-Cambiar el directorio de trabajo (opcional): 
Si el archivo del código está en una carpeta diferente a la que aparece en la terminal, 
navega hasta el directorio donde se encuentra el archivo. Puedes usar el comando cd
seguido de la ruta completa del directorio para cambiar el directorio de trabajo.

5-Ejecutar el script: 
Una vez que estés en el directorio correcto, simplemente escribe el siguiente comando en 
la terminal y presiona "Enter":

                          python script_crud.py

Reemplaza "script_crud.py" con el nombre del archivo que contenga tu código Python. 
Presionar "Enter" iniciará la ejecución del script.

6-Interactuar con el script: 
Ahora, el script se estará ejecutando y verás el menú en la terminal, lo que te permitirá 
seleccionar diferentes opciones para realizar las operaciones CRUD o hacer una copia de 
seguridad de la base de datos SQLite. Sigue las instrucciones que aparecen en la terminal 
para interactuar con el script.

Al ejecutar el script automaticamente se creara un archivo de bases de datos de tipo "db"
en el mismo directorio donde guardo el código. Si lo borras por accidente, no pasa nada
ya que al volver a ejecutar el script se volverá a crear la base de datos.

Al seleccionar la opcion de crear backup de bases de datos, automaticamente se generará una
copia de seguridad de la bases de datos en el mismo directorio donde guardo el código. 
Si lo borras por accidente, no pasa nada ya que al volver a ejecutar dicha opcion se volverá 
a crear la copia de seguridad

7-Finalizar la ejecución: 
Para finalizar la ejecución del script, simplemente selecciona la opción "6" en el menú y 
presiona "Enter" o cierra la terminal.
